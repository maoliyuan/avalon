# 阿瓦隆记牌器

一个**纯前端、可离线**的阿瓦隆私人记牌 / 身份推理工具。为「线下只有一台 iPhone」的场景做的：装到主屏当独立 App 用，数据只存在你自己手机本地。

## 功能

- **身份标记（核心）**：对每个号码（1–12）记录你的身份猜想。每条 = `确定度 × 身份`，四档确定度：
  - 确定是 / 可能是 / 可能不是 / 确定不是
  - **可叠加**：同一号码可挂多条（如 3 号：`可能是梅林` + `确定不是坏人`）。点标记即删。
  - 可标记的身份清单在 [`app.js`](app.js) 顶部的 `IDENTITIES` 数组里，随手增删。
- **每轮记录**：每轮记下 队长 / 提名队员 / 每个号码的赞成·反对。纯客观记录，**不做任何推算，不给你 bias**。
- **本地存储 + 离线**：首次联网打开一次后即可全程离线；关掉重开数据还在。
- **新开一局 / 回溯上一局**：新开一局会把当前记录存为「上一局」；「回溯上一局」在这一局和上一局之间来回切换（只保留一层）。

## 部署到 GitHub Pages

1. 新建一个仓库（比如叫 `avalon`），把本目录所有文件传上去（`index.html` 必须在仓库根目录）。
2. 仓库 **Settings → Pages → Build and deployment**：Source 选 `Deploy from a branch`，分支选 `main`、目录选 `/ (root)`，保存。
3. 等一两分钟，页面地址是 `https://<你的用户名>.github.io/<仓库名>/`。

> 全部用相对路径写的，放在子路径（`/仓库名/`）下也能正常离线，无需改动。

## iPhone 上安装

1. 用 **Safari** 打开上面的地址（必须 Safari，Chrome 不行）。
2. 点底部「分享」→ **添加到主屏幕**。
3. 之后从主屏图标打开：全屏、无地址栏，跟原生 App 一样，且**可离线使用**。

数据存于该网站的本地存储中。装到主屏后系统一般会长期保留；只要别在设置里清除该网站数据或删掉图标，记录就一直在。

## 更新版本

改完代码后，把 [`sw.js`](sw.js) 里的 `VERSION` 加一（`v1` → `v2`…）再推送。各设备下次联网打开时会自动拉到新版。

## 本地预览

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

（Service Worker 需要 http/https 环境，直接双击打开 `file://` 不会启用离线，但功能可用。）

## 文件说明

| 文件 | 作用 |
|---|---|
| `index.html` | 页面骨架 |
| `styles.css` | 样式（移动端优先，深色） |
| `app.js` | 全部逻辑与数据模型 |
| `manifest.webmanifest` | PWA 清单（图标、独立窗口） |
| `sw.js` | Service Worker（离线缓存） |
| `icon-*.png` / `apple-touch-icon.png` | 应用图标 |
